Background Information on AWS Redshift

Link: AWS Redshift Docs
Amazon Redshift is a fast, fully managed, petabyte-scale data warehouse that makes it simple and cost-effective to analyze all your data using your existing business intelligence tools.


Instructions to Launch your Redshift Cluster

1. Fill out the Cluster Launch Form and press Submit
2. Put in your AWS key and secret into the respective variable fields
3. Ask your admin to create a VPC security group and send the ID (e.g sg-3a907211)
4. Put in your VPC id into the respective variable field
5. Go back to the Agility Platform UI and run the "Cluster Launch" Operational Script


How to Access your Redshift Environment

After a few minutes, the operational script will run. The script will yield an endpoint.
Use the following instructions to access the endpoint - Redshift Connect Info
Download either the JDBC or ODBC Driver. The links below contain more instructions and links to the downloads
Use either the SQL Workbench or a Partner Tool to connect to Redshift
